**Status Sprint Goal:** ✅ Tercapai  
**Sprint Goal:** *"Membangun halaman login dan logout untuk semua role pengguna, serta membangun dashboard monitoring produksi untuk owner dengan tampilan responsif."*

**A. Yang Berjalan dengan Baik**

1. Daily Scrum berjalan lancar menggunakan Discord, komunikasi tim menjadi rutin dan terstruktur.  
2. Pembuatan repositori terpisah untuk Frontend dan Backend di GitHub membuat manajemen kode lebih rapi.  
3. Dokumen testing mudah dibuat karena materi sudah dipelajari di mata kuliah lain, sehingga QA dapat bekerja lebih cepat.  
4. API berhasil setelah koordinasi intensif dengan tim Backend, fondasi komunikasi teknis mulai terbangun.  
5. Koordinasi tim berjalan baik melalui WhatsApp dan pertemuan offline, termasuk penjadwalan Scrum events.

**B. Yang Tidak Berjalan dengan Baik**

1. **Koneksi Laravel ke MySQL** error dan memiliki kendala teknis.  
2. **Menyambungkan codingan ke GitHub** terasa sulit.  
3. **Penjadwalan pembuatan DoD** terlambat, baru dilakukan di tengah Sprint.  
4. **Validasi requirement dilakukan di akhir (big bang)** , bukan bertahap.  
5. **Culture shock sebagai BA** menyebabkan khawatir knowledge tidak terdistribusi ke tim developer.  
6. **Kebutuhan sering berubah** sehingga use case harus sering di-update.  
7. **Manajemen waktu** pengerjaan cenderung mepet di akhir Sprint.
